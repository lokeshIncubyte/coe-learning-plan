# Day 7 Todo App

React + TypeScript Todo app built with Vite.

## Run

From `day7/todo-app`:

```powershell
npm install
npm run dev
```

## Test

```powershell
npm test
```

## Lint and Build

```powershell
npm run lint
npm run build
```

## Scope

- Add todo
- Toggle complete
- Delete todo
- Edit todo
- Filter and search behavior

## Development Approach

- Test-first workflow (RED -> GREEN -> REFACTOR)
- Small incremental commits
